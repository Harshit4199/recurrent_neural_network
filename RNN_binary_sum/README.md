# RNN Demo

Overview
============
This project helps train a Recurrent neural net to predict the sum of two binary numbers.

Dependencies
============
* numpy
* copy

Use [pip](https://pypi.python.org/pypi/pip) to install any missing dependencies

Basic Usage
===========

Run rnn.py, that's it!
